# mochi-store

### 如何啟動專案？

1. `git fetch`
2. `git pull`
3. 到專案根目錄 (mochi-store 資料夾) 
4. `npm install`
5. 用 `npm test` 來啟動 Express 伺服器
6. 到瀏覽器裡面輸入 http://localhost:3000 來查看專案
7. 開始 coding
8. 之後要更新專案的時候，記得都操作以下指令：

   ```git
   git fetch
   git status
   ```

   a. 基礎操作：執行到 `git status` 時出現 "can be fast-forwarded" 的時候，就可以直接執行 `git pull` 做專案合併

   b, 如果出現衝突，就要逐項更改。
